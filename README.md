# front

### Vite + React + TailwindCSS v4 + Redux Toolkit
### Node 22.13.0

## 개발 시작

```sh
npm -g install yarn
yarn or yarn install
yarn dev
```

## Env

## 주요 폴더 구조
- src/components: 공통 컴포넌트
- src/pages: 페이지 컴포넌트
- src/store: Redux 상태관리
- src/styles: 글로벌 스타일 및 Tailwind

## 주요 라이브러리
- Vite
- React
- TailwindCSS v4
- Redux Toolkit
- ESLint, Prettier
